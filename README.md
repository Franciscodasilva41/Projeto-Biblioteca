# Sistema de Biblioteca em Java.
Projeto desenvolvido em Java utilizando JDBC e MySQL para gerenciamento de uma biblioteca.

# Funcionalidades 

- Cadastro de usuários
- Cadastro de livros
- Listagem de livros e usuarios
- Atualizações e remoção
- Sistema de empréstimo de livros
  
#Tecnologias utilizadas
- Java
- JDBC
- Swing(interface grafica)

# Estrutura do projeto
- model -> classes(Livro, Usuario)
- dao -> acesso ao banco(LivroDAO, UsuarioDAO)
- service -> regras de negócio
- view -> interface gráfica

 # Como executar
 1. Configurar o banco MySQL
 2. Rodar o projeto no NetBeans
 3. Executar a classe Main
